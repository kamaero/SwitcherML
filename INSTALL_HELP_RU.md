# SwitcherLM — Установка для тестирования

Ниже два сценария: для вас (сборка `.pkg`) и для пользователей, которым вы отправляете установщик.

## 1) Как собрать `.pkg` у себя

Из корня проекта:

```bash
./scripts/sync-version.sh
./scripts/package.sh
```

Результат:

- установщик появится в папке `dist/`
- имя файла: `SwitcherLM-<версия>.pkg` (например `SwitcherLM-1.0.1.pkg`)

Опции:

```bash
./scripts/package.sh --debug
./scripts/package.sh --verbose
```

Примечание по версии:

- версия синхронизируется с изменениями кода (`Sources/SwitcherLM`), а не с количеством сборок
- повторная сборка одного и того же коммита не увеличивает версию

## 2) Как установить пользователю (macOS)

### Шаг 1. Запустить установщик

1. Скачайте `SwitcherLM-*.pkg`.
2. Дважды кликните по файлу и пройдите шаги установщика.
3. Приложение установится в `/Applications/SwitcherLM.app`.

Если macOS блокирует запуск пакета ("developer cannot be verified"):

1. Откройте `System Settings -> Privacy & Security`.
2. Внизу нажмите `Open Anyway` для SwitcherLM installer.
3. Повторите запуск `.pkg`.

### Шаг 2. Первый запуск приложения

1. Откройте `/Applications/SwitcherLM.app`.
2. Если macOS блокирует первый запуск приложения, откройте его через правый клик -> `Open`.
3. В меню-баре появится иконка SwitcherLM.

### Шаг 3. Выдать доступ Accessibility (обязательно)

Без этого приложение не сможет отслеживать/исправлять ввод.

1. Откройте `System Settings -> Privacy & Security -> Accessibility`.
2. Включите `SwitcherLM`.
3. Если пункта нет в списке:
   - нажмите `+`
   - выберите `/Applications/SwitcherLM.app`
   - включите тумблер

После выдачи доступа иногда нужно перезапустить приложение.

## 3) Обновление до новой версии

Просто запустите новый `.pkg` поверх старой версии. Удалять старую вручную не требуется.

## 4) Удаление приложения

1. Закройте SwitcherLM (через иконку в меню-баре -> Quit).
2. Удалите `/Applications/SwitcherLM.app`.

При желании можно очистить пользовательские данные:

- `~/Library/Application Support/SwitcherLM/`
- настройки в `UserDefaults` для bundle id `com.switcherlm.app`
